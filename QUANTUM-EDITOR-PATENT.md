## PATENT CLAIMS - PROCEDURAL DESIGN ELEMENTS

### 1. SYNTAGMATIC BLOCK SYSTEM
**What it is**: Each video block = syntagma (meaningful unit)

**Properties**:
- Temporal range (start/end timestamps)
- Thematic category (Media Apparatus, Systemic Critique, etc.)
- Narrative weight (affects physics behavior)
- Color coding (visual semantic grouping)

**How it works**: Physical adjacency creates syntagmatic chains (Eisenstein montage theory)

**Patent Claim**: 
> "Method for representing video segments as spatial objects with semantic properties that generate meaning through physical proximity, wherein each object contains temporal, thematic, and behavioral metadata that influences its interaction with adjacent objects"

### 2. SPATIAL PATTERN MATCHING (Match-3 Mechanics)
**What it is**: Collect fragments only when 3+ identical themes align spatially

**How it works**:
- System analyzes completed lines for thematic patterns
- Only concepts appearing 3+ times are "collected"
- Pattern recognition determines editorial permission

**Patent Claim**:
> "System for extracting media segments based on threshold-based spatial pattern recognition within constraint-based puzzle mechanics, wherein editorial access requires minimum pattern density of semantically identical elements"
